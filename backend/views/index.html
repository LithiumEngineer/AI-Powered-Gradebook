WorkSheeps API Documentation


Instructions for GET /students Route

Retrieve all students.

Instructions for GET /students/:id Route

Retrieve a student by their Teacher ID.

  • Replace :id in the route with the actual Teacher ID.

Instructions for POST /students Route

Create a new student.

  • Send a POST request to /students route.
  • The body of the request should contain a JSON object with the following properties:
    • {first_name, last_name, teacher_id}

Instructions for PUT /students Route

Update an existing student.

  • Send a PUT request to /students route.
  • The body of the request should contain a JSON object with the following properties:
    • {record_id, first_name, last_name, teacher_id}

Instructions for DELETE /students Route

Delete a student by their record ID.

  • Send a DELETE request to /students route.
  • The body of the request should contain a JSON object with the following properties:
    • {record_id}

Instructions for GET /tests Route

Retrieve all tests.

Instructions for GET /tests/:id Route

Retrieve a test by Owner ID.

  • Replace :id in the route with the actual Owner ID.

Instructions for POST /tests Route

Create a new test.

  • Send a POST request to /tests route.
  • The body of the request should contain a JSON object with the following properties:
    • {topic, student_grades_json, test_type, owner_id}

Instructions for PUT /tests Route

Update an existing test by its record ID.

  • Send a PUT request to /tests route.
  • The body of the request should contain a JSON object with the following properties:
    • {record_id, topic, student_grades_json, test_type, owner_id}

Instructions for DELETE /tests Route

Delete a test by its record ID.

  • Send a DELETE request to /tests route.
  • The body of the request should contain a JSON object with the following properties:
    • {record_id}
